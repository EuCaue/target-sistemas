# Resposta 3

## Pergunta

Observe o trecho de código abaixo:

```c
int INDICE = 13, SOMA = 0, K = 0;
Enquanto K < INDICE faça { K = K + 1; SOMA = SOMA + K; }
Imprimir(SOMA);
```

Ao final do processamento, qual será o valor da variável `SOMA`?

---

## Resposta

O valor final da variável `SOMA` será de **91**.
